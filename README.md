# ISRP_R_SOP

This repo contains the files required to view and use the ISRP Standard Operating Procedure (SOP) for R and RStudio. The SOP contains a detailed tutorial for using R and RStudio for PCB analyses.

To run the R Markdown file, download all files into the same directory. Next open the R project file in RStudio, then open R_SOP_tutorial.Rmd. You can interact with the Markdown code directly and run the code provided. To view the SOP as an HTML page, click on "knit" near the top of the scripts panel.
